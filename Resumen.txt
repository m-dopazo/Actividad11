Una API es un conjunto de protocolos que se usa para diseñar e integrar el software de las aplicaciones.

Las API permiten que sus productos y servicios se comuniquen con otros, sin necesidad de saber cómo están implementados. Pueden simplificar la forma en que los desarrolladores integran los elementos de las aplicaciones nuevas en una arquitectura actual. También permiten compartir datos con usuarios externos.

Políticas de lanzamiento de las API:
    -Privado: Las API solo se pueden usar internamente, así que las empresas tienen un mayor control sobre ellas.
    -De partners: Las API se comparten con partners empresariales específicos
    -Público: Todos tienen acceso a las API, así que otras empresas pueden desarrollar API que interactúen con las de usted

Innovaciones con las API
Las API compartidas, ya sea con los partners elegidos o con todo el mundo, tienen efectos positivos. Mientras más se asocie con otros gracias a las API, mayor difusión obtendrá su marca
Hacer pública la tecnología da resultados novedosos e inesperados que a veces alteran sectores completos

Las API remotas
Las API remotas están diseñadas para interactuar en una red de comunicaciones.la mayoría de las API están diseñadas de acuerdo con los estándares web.Por lo general, las API web usan HTTP para los mensajes de solicitud y proporcionan una definición de la estructura de los mensajes de respuesta. Estos mensajes suelen tomar la forma de un archivo XML o JSON
